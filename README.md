# 📊 E-Commerce Sales Dashboard

## Overview
This project analyzes e-commerce sales data and presents key business insights through an interactive Power BI dashboard. The dashboard helps monitor sales performance, profit, orders, payment methods, and product categories.

## Objectives
- Analyze sales and profit performance
- Track total orders and average order value
- Compare category-wise performance
- Understand payment method distribution
- Create an interactive dashboard for business decision-making

## Tools Used
- Microsoft Excel (Data Preparation)
- Power BI (Dashboard & Visualization)

## Dashboard Features
- Total Sales
- Total Profit
- Total Orders
- Average Order Value (AOV)
- Category-wise Profit Analysis
- Payment Method Distribution
- Order Analysis by Category

## Key Insights
- Identified top-performing product categories.
- Compared profit across different categories.
- Analyzed customer payment preferences.
- Monitored overall sales and business performance.
